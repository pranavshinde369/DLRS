# Examples

## 📌 Overview
This directory contains example archive templates used in the DLRS system.

All examples use **fictional data and placeholder content**.
No real or sensitive information is included.

---

## 🧠 Purpose of these examples
These examples are designed to:
- Help beginners understand how DLRS archives are structured  
- Demonstrate different use cases (e.g., personal, public, protected data)  
- Provide a starting point for creating your own archive  

---

## 📂 What you will find here
Each example folder contains:
- A `manifest.json` file  
- Metadata describing the archive  
- Structured sample data  

---

## 💡 Tips for contributors
- Keep examples simple and easy to understand  
- Use fictional data only  
- Add comments to explain important fields
